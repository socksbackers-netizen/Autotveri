# Telegram Firebase Bot

Multi-Firebase management bot for device control.

## Commands
- /addfirebase <url> [name] - Add Firebase
- /listfirebase - List all
- /usefirebase <id> - Switch
- /removefirebase <id> - Remove
- /startmonitor - Start monitoring
- /stopmonitor - Stop monitoring
- /sendall <msg> - Send to all devices
- /status - Status
- /help - Help

## Deploy on Render
1. Fork this repo
2. Create new Web Service on Render
3. Set BOT_TOKEN environment variable
4. Deploy!
